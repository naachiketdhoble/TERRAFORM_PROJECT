# Terraform AWS Infrastructure Project

This project uses Terraform to create AWS infrastructure resources including:

- VPC
- Subnet
- Internet Gateway
- Route Table
- Security Group
- IAM Role
- IAM Policy Attachments
- IAM Instance Profile
- EC2 Instance

## Technologies Used
- Terraform
- AWS
- Ubuntu Linux
- Git & GitHub

## AWS Services Used
- EC2
- VPC
- IAM
- Security Groups

## Files
- `ec2.tf` → Main Terraform configuration
- `.gitignore` → Ignore Terraform state and sensitive files
- `README.md` → Project documentation
